# write-zenn-article スキル

Zennテックブログ記事を自動生成するスキルです。ユーザーの既存記事スタイルを分析し、それに沿った下書きを作成します。

## 概要

このスキルは、トピック名を指定するだけで、Zennの記事フォーマットに準拠した下書きを自動生成します。

主な機能：
- ユーザーの記事スタイルを踏襲したFront Matter生成
- 典型的な記事構成の自動生成（導入 → 解説 → 検証 → まとめ）
- カジュアルで親しみやすい文体
- コードブロックのプレースホルダー配置
- zenn-cliによる記事ファイル作成

## 使用方法

```bash
# 基本的な使用方法
/write-zenn-article Go 1.24の新機能

# 引数なし（対話形式）
/write-zenn-article

# 複数単語のトピック
/write-zenn-article OpenTelemetryの活用方法
```

## ディレクトリ構成

```
write-zenn-article/
├── SKILL.md                    # スキル定義（実装手順と詳細）
├── README.md                   # このファイル
├── templates/
│   └── article_template.md    # 記事テンプレート
└── examples/
    └── example_output.md      # 出力例
```

## ファイルの役割

### SKILL.md

スキルの実装詳細を記載したメインドキュメントです。以下の情報を含みます：

- 入力形式（引数の解析方法）
- ユーザーの記事スタイルの特徴
- 実行フロー（Phase 1〜5）
- Front Matter生成ルール
- 記事構成パターン
- エッジケース対応
- 成功基準

### templates/article_template.md

記事の基本テンプレートです。スキルが生成する記事の構成とスタイルの参考になります。

### examples/example_output.md

スキルが生成する記事の具体例です。以下のトピックで作成したサンプル：

- トピック: Go 1.24のイテレータ
- Front Matter: title, emoji, type, topics
- 記事構成: 導入 → 解説 → 実装 → ベンチマーク → まとめ

## 生成される記事の特徴

### Front Matter

```yaml
title: "{トピック名}を試してみた！"
emoji: "🚀"
type: "tech"
topics:
  - "{主要技術}"
  - "{関連キーワード}"
published: false
```

### 記事構成

1. **導入**: 背景・きっかけ
2. **対象**: 想定読者
3. **解説**: 技術の詳細
4. **実装**: コード例
5. **検証**: ベンチマーク結果
6. **まとめ**: 学んだこと

### 文体

- カジュアルな「です・ます」調
- 適度な絵文字使用（🚀、💡、🙌など）
- 読者への語りかけ
- 探求的な表現（「〜してみた」「〜を調べてみた」）

## ユーザーの既存記事スタイル

既存記事18件の分析結果：

- **主要テーマ**: Go言語（10/18記事）、新機能解説、内部実装の深掘り
- **対象読者**: 中〜上級エンジニア
- **アプローチ**: 実証的（コード例 + ベンチマーク）
- **深さ**: 公式ドキュメント、GitHub、内部実装レベル

## 出力ファイル

**保存先**: `~/repos/zenn/articles/{slug}.md`

**スラッグ**: zenn-cliのランダム生成（12文字）

## エラーハンドリング

| エラー | 対応 |
|--------|------|
| トピックが空 | AskUserQuestion でトピックを入力 |
| zenn-cli が利用不可 | 直接Markdownファイルを作成 |
| ディレクトリが存在しない | エラーメッセージとパス確認を促す |
